# Case Técnico — Painel de Projetos (PVT Software & Serviços)

Case técnico para a vaga de **Desenvolvedor Web e Infraestrutura JR** na PVT Software & Serviços (parceira TOTVS).

O objetivo é projetar tecnicamente o módulo **"Painel de Projetos"** do portal de gestão da PVT: modelagem de domínio, arquitetura, design de API e regras de negócio, considerando que o dado-fonte vem de um ERP externo (TOTVS RM), tratado aqui como uma caixa externa mockada.

## Estrutura da entrega

- **Parte 1 — Design e Arquitetura** (obrigatória): documento cobrindo modelagem do domínio, arquitetura, design da API, regras de negócio, trade-offs e visão de evolução/escala.
- **Parte 2 — Implementação** (bônus): fatia mínima do que foi desenhado.
